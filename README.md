一、项目概述
 
1.1 项目介绍
 
本项目是一款基于前端原生技术开发的本地双人五子棋小游戏，全程无需服务器、无需联网、无广告、无数据上传，纯本地浏览器运行。游戏支持双人同屏轮流对战，实现标准五子棋完整规则，包含落子判定、胜负智能检测、禁止违规操作、对局重置、状态提示等全套核心功能。
 
项目代码轻量化、运行高效、兼容性强，打开即可游玩，适合课堂实训、课程设计、期末作业、入门项目展示。
 
1.2 开发技术栈
 
- HTML5：搭建棋盘结构、页面布局、游戏主体框架
- CSS3：实现棋盘样式、棋子样式、hover效果、界面美化
- 原生 JavaScript：实现所有游戏核心逻辑（落子、判赢、状态更新、交互校验）
- 运行环境：任意PC浏览器（Chrome、Edge、Firefox、360浏览器等）
 
1.3 项目特点
 
1. 纯本地离线运行：零网络请求、零后台接口、所有逻辑前端完成
2. 操作简洁友好：点击即可落子，新手零学习成本
3. 严格规则校验：禁止重复落子、禁止棋盘外落子，杜绝违规操作
4. 智能全局判赢：自动检测横向、纵向、左斜、右斜四个方向五连子
5. 实时状态提示：页面实时显示当前落子方、胜利提示、对局结束状态
6. 轻量化高性能：代码精简、无冗余，运行流畅不卡顿
7. 可拓展性强：结构清晰，方便后续添加AI、悔棋、计时、计分等功能
 
二、游戏功能详细说明
 
2.1 基础对局功能
 
1. 双人轮流落子
默认黑棋先手，黑白棋子交替落子，落子后自动切换落子方，保证对局顺序规范。
2. 合法落子校验机制
 
- 已落子位置禁止重复点击，重复点击无任何响应
- 棋盘空白交叉点才可落子
- 棋盘区域外点击无效，防止误操作
 
2.2 智能胜负判断系统
 
程序每一次落子后，自动触发全局胜负检索，遍历当前落子点的四个方向：
 
- 水平横向（左右）
- 垂直纵向（上下）
- 左上—右下斜向
- 右上—左下斜向
 
统计同色连续棋子数量，满足五子连珠即刻判定胜利，弹出胜利提示并终止当前对局，禁止继续落子。
 
2.3 平局判定
 
当整个 15×15 棋盘全部落满棋子，且无任何一方达成五连子，自动形成平局，对局结束，可手动重置新局。
 
2.4 对局重置功能
 
内置重新开始功能按钮，点击后可：
 
- 清空全部棋盘棋子
- 重置落子顺序，恢复黑棋先手
- 清空胜负状态，开启全新对局
 
2.5 实时状态显示
 
页面顶部实时展示游戏状态：
 
- 提示「黑棋回合」「白棋回合」
- 胜利后提示「黑棋获胜！」「白棋获胜！」
- 对局结束锁定棋盘，禁止继续落子
 
三、游戏规则详细介绍
 
1. 本游戏采用标准 15×15 专业五子棋棋盘
2. 对局规则：黑先白后，交替落子，每次仅落一子
3. 胜负规则：任意玩家率先在横、竖、斜向任意一条直线连成五颗同色棋子即获胜
4. 落子不可移动、不可删除（原版基础功能）
5. 棋盘填满且无五连子则判定平局
6. 对局结束后必须重置方可继续游戏
 
四、系统运行说明
 
4.1 运行环境要求
 
- 操作系统：Windows / MacOS / Linux 全平台兼容
- 运行软件：任意现代浏览器
- 无需安装软件、无需配置环境、无需数据库、无需服务器
 
4.2 启动方式
 
1. 下载全部项目文件，保证目录完整
2. 直接双击根目录  index.html  文件
3. 浏览器自动打开游戏界面，即刻开始双人对战
 
4.3 操作步骤
 
1. 打开页面后默认新局，黑棋先行
2. 两名玩家轮流点击棋盘交叉点落子
3. 系统自动判断胜负，获胜后弹窗提示
4. 点击重新开始，即可开启下一局对战

六、核心逻辑原理说明
 
6.1 棋盘数据存储
 
通过二维数组 15×15模拟真实棋盘：
 
- 0：空位
- 1：黑棋
- 2：白棋
 
每次落子同步更新数组数据，所有胜负检测基于数组数据遍历判断，精准、无误差。
 
6.2 落子逻辑流程
 
1. 玩家点击棋盘位置
2. JS 计算点击对应棋盘坐标
3. 判断该坐标是否为空
4. 为空则渲染对应颜色棋子，并更新数组
5. 不为空则拦截操作，无响应
6. 落子成功后自动执行判赢函数
 
6.3 判赢核心原理
 
以当前最新落子点为中心，向四个正反方向延伸检索：
 
1. 统计同色相连棋子总数
2. 只要任意方向连续数量 ≥5，判定胜利
3. 终止对局、锁定棋盘、弹出提示
 
只检索当前落子周边，而非全局遍历，算法高效、性能优异。
